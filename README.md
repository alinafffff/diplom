ТРЕБОВАНИЯ:

1) JDK 17+ (для серверной части)
2) Maven (для сборки бэка)
3) PostgreSQL 14+ (для базы данных)
4) Flutter SDK: >=3.3.0 <4.0.0 (для запуска клиентской части)
5) Приложение VS Code (для удобного запуска Flutter приложения)
6) Среда разработки для запуска Java-приложения (например, Intellij idea)
7) PostgreSQL, если еще не установлен.
8) Подключение к интернету

РАЗВОРАЧИВАНИЕ:
1) Скачайте из папки 212053 папки diplom и inpit_front. Это серверная и клиентская части приложения.
2) Создайте базу данных inpit командой:
createdb inpit
3) Восстановите базу данных из дампа командой:
psql -U postgres -d inpit -f путь_к_файлу/inpitDB.sql
4) Запустите проект diplom в Среде разработки Intellij ieda. 
Серверная часть будет доступна по адресу http://localhost:8080.
5) Запустите проект inpit_front. Для этого Откройте папку /frontend в терминале.
Установите зависимости командой:
flutter pub get
Запустите приложение на физическом устройстве:
flutter run
